# Barbera d'Alba import-prijs ontleed — wat kost een fles Vinitho écht

Eén fles, alle kosten uit elkaar getrokken. Geen verkooppraat, alleen
de cijfers zoals ze door mijn boekhouding gaan.

## De fles

Cascina delle Rose Barbera d'Alba 2022. Piemonte, Italië. Producent
is familiebedrijf in Barbaresco, ±3 hectare, biologisch sinds 2018.

## Inkoop-prijsopbouw (per fles)

| Post | Bedrag | % van verkoop |
|---|---|---|
| Inkoop wijnonder EXW Barbaresco | €4,85 | 32% |
| Transport IT → NL (geconditioneerd) | €0,40 | 3% |
| Accijns + import-BTW-correctie | €0,89 | 6% |
| Verpakking + etiket + kurk | €0,25 | 2% |
| **Subtotaal kostprijs** | **€6,39** | **43%** |

## Verkoop-prijsopbouw

| Post | Bedrag | % van verkoop |
|---|---|---|
| Verkoopprijs ex-BTW | €13,72 | 92% |
| BTW 9% | €1,23 | 8% |
| **Verkoopprijs incl-BTW** | **€14,95** | **100%** |

## Marge

- **Brutowinst** per fles: €14,95 − €6,39 = **€8,56** = 57% van verkoop
- **Netto na logistiek + opslag**: ~€7,80 = 52%
- Dit is vóór belasting + eigenaar-uur

Bij 100 flessen/maand op deze ene wijn = €780/maand bruto winst.

## Wat je hiervan kunt leren

1. **Italiaanse wijn is margemateriaal** voor een eenmanszaak — geen
   supermarkt concurreert op niche-producenten
2. **EXW-prijs is niet de eindprijs** — reken altijd 30-40% bovenop
   voor transport, accijns, verpakking
3. **BTW-tarief 9% geldt voor wijn** (niet 21%) — controleer dit per
   productcategorie
4. **Familie-producenten accepteren kleine orders** — geen
   minimumpallet van 600 flessen nodig

## Bron

Eigen administratie Q2 2026. BTW-tarief gecontroleerd bij Belastingdienst.

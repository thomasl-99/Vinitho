# Vinitho — landingspagina

Deze landingspagina is onderdeel van de AI-vindbaarheid-implementatie
door LA Notes (KvK 97388505, btw NL005266959B81).

## Wat hier staat

- `index.html` — landingspagina met JSON-LD structured data
- `llms.txt` — AI-specifieke samenvatting voor ChatGPT/Perplexity/Claude
- `robots.txt` — 16 AI-crawlers toegelaten
- `_posts/` — blog-posts (3+ aanbevolen)
- `.github/workflows/` — automatische GitHub Pages deploy

## Volgende stappen

1. Open elk bestand in `_posts/` en vervang de placeholders
2. Activeer GitHub Pages: Settings → Pages → Source: GitHub Actions
3. (Optioneel) Koppel custom domain `vinitho.nl`
4. Dien `index.html` in bij Google Search Console
